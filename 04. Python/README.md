# 🐍 Python Projects Collection

Bienvenue dans mon dépôt regroupant l’ensemble de mes projets développés avec **Python**.

Ce repository met en avant ma maîtrise du langage à travers différents cas d’usage : **scripts, analyse de données, web scraping et manipulation de datasets**, tout en explorant des bibliothèques clés de l’écosystème Python.


## 📁 Structure du Repository

Chaque dossier contient :

* Des scripts Python
* Des exemples pratiques
* Des datasets (si nécessaires)
* Des explications ou notebooks


## 🛠️ Technologies & Concepts Utilisés

Les projets couvrent plusieurs aspects du développement en Python :

* **Python (Core)**
* **Programmation procédurale et modulaire**
* **Manipulation de fichiers (CSV, JSON, TXT)**
* **Web Scraping avec BeautifulSoup**
* **Analyse de données avec Pandas**
* **Nettoyage et transformation de données**
* **Automatisation de tâches**
* **Jupyter Notebook (pour les tutos et analyses)**


## 🌐 Web Scraping

Une partie du repo est dédiée au scraping web avec **BeautifulSoup** :

* Extraction de données depuis des pages HTML
* Parsing et navigation dans le DOM
* Récupération de contenus (titres, prix, liens, etc.)
* Nettoyage et structuration des données extraites
* Export vers CSV ou JSON


## 📊 Pandas Tutorials

Des tutoriels progressifs pour maîtriser **Pandas** :

* Chargement de datasets (CSV, Excel)
* Exploration des données (`head`, `info`, `describe`)
* Filtrage, tri et sélection
* GroupBy et agrégations
* Gestion des valeurs manquantes
* Nettoyage et transformation
* Création de colonnes calculées
