# 📊 Tableau Data Visualization

Ce repository regroupe mes projets réalisés avec **Tableau**, mettant en avant mes compétences en **visualisation de données**, **analyse interactive** et **création de dashboards décisionnels**.

Il illustre ma capacité à transformer des données brutes en **insights clairs, visuels et exploitables**.


## 🎯 Objectifs du Repository

* Créer des visualisations interactives et impactantes
* Explorer et analyser des datasets réels
* Construire des dashboards orientés prise de décision
* Maîtriser les fonctionnalités avancées de Tableau
* Développer des compétences en **Data Storytelling**


## 🛠️ Compétences & Concepts Couverts

### 📌 Data Visualization

* Graphiques (barres, lignes, cartes, scatter plots)
* Visualisation des tendances et KPIs
* Choix des visualisations adaptées aux données


### 📌 Dashboards Interactifs

* Création de dashboards dynamiques
* Filtres, paramètres et actions
* Navigation fluide entre les vues


### 📌 Data Analysis

* Exploration des données (EDA)
* Identification de patterns et tendances
* Analyse comparative


### 📌 Calculated Fields

* Création de champs calculés
* Logique conditionnelle
* Agrégations personnalisées


### 📌 Data Cleaning & Preparation

* Connexion aux sources de données
* Transformation et structuration
* Gestion des types de données


### 📌 Data Storytelling

* Mise en contexte des données
* Construction de narrations visuelles
* Communication claire des insights


## 📁 Structure du Repository

```bash
📦 tableau-projects
 ┣ 📂 project-1/
 ┃ ┣ dataset.csv
 ┃ ┣ dashboard.twbx
 ┃ ┗ README.md
 ┣ 📂 project-2/
 ┗ 📄 README.md
```

Chaque projet contient :

* Le dataset (ou lien vers la source)
* Le fichier Tableau (`.twb` ou `.twbx`)
* Une description du projet
