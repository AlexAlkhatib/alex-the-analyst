# 📊 Alex The Analyst

Ce repository regroupe des projets réalisés dans le cadre de la formation **“The Data Analyst Bootcamp”** proposée par le YouTuber **Alex The Analyst**.

Les projets couvrent plusieurs outils clés de la Data :

* **Excel**
* **Tableau**
* **Power BI**
* **Python**


## 📌 Note

⚠️ Ces projets ne sont pas des créations originales.
Ils sont directement inspirés et reproduits à partir du contenu pédagogique de la formation, dans un objectif d’apprentissage et de pratique.


## 🎯 Objectif

✔️ Mettre en pratique les concepts de Data Analysis
✔️ Maîtriser différents outils utilisés en entreprise
✔️ Consolider mes compétences à travers des cas concrets
