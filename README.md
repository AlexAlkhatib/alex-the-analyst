# 📊 Excel Data Analysis

Ce repository regroupe mes projets et exercices en **analyse de données avec Microsoft Excel**, couvrant les techniques essentielles de manipulation, d’analyse et de visualisation de données.

Il met en pratique les outils fondamentaux d’Excel, depuis le **nettoyage des données** jusqu’à la création de **dashboards interactifs**.


## 🎯 Objectifs du Repository

* Maîtriser les fonctionnalités clés d’Excel pour l’analyse de données
* Manipuler et transformer des datasets réels
* Construire des visualisations claires et pertinentes
* Développer des dashboards interactifs pour la prise de décision
* Consolider mes compétences en **Data Analysis**


## 🛠️ Compétences & Concepts Couverts

### 📌 Pivot Tables

* Création de tableaux croisés dynamiques
* Agrégation et segmentation des données
* Analyse rapide de grands volumes de données


### 📌 Formules Essentielles

* `SUM`, `AVERAGE`, `COUNT`, `IF`
* `VLOOKUP` / `HLOOKUP`
* Fonctions logiques et statistiques


### 📌 INDEX & MATCH

* Recherche avancée de données
* Alternative plus flexible à `VLOOKUP`
* Combinaison de fonctions pour des requêtes complexes


### 📌 Data Cleaning

* Suppression des doublons
* Gestion des valeurs manquantes
* Standardisation des formats
* Transformation et structuration des données


### 📌 Conditional Formatting

* Mise en évidence des tendances
* Règles basées sur des conditions
* Visualisation rapide des anomalies


### 📌 Excel Charts

* Graphiques dynamiques (barres, lignes, camemberts, etc.)
* Visualisation des tendances et KPIs
* Création de rapports visuels


## 🚴‍♂️ Projet Final : Bike Sales Analysis

### 📊 Description

Projet complet d’analyse de données basé sur un dataset de ventes de vélos.

Ce projet résume l’ensemble des compétences acquises, en suivant un workflow complet :

### 🔄 Étapes réalisées

1. **Data Cleaning**

   * Nettoyage et préparation des données
   * Suppression des incohérences et valeurs manquantes

2. **Data Transformation**

   * Création de nouvelles colonnes
   * Structuration du dataset

3. **Exploratory Data Analysis (EDA)**

   * Analyse des tendances de ventes
   * Identification des facteurs influents

4. **Pivot Tables**

   * Analyse des ventes par catégorie, âge, revenu, etc.

5. **Visualisation**

   * Création de graphiques clairs et pertinents

6. **Dashboard Interactif**

   * Mise en place de filtres (slicers)
   * Visualisations dynamiques
   * Interface utilisateur intuitive


### 📈 Résultat

* Dashboard interactif permettant d’analyser :

  * Le profil des acheteurs
  * L’impact du revenu et de l’âge
  * Les tendances de vente

* Outil prêt pour **la prise de décision business**
