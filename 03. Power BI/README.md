# 📊 Power BI Projects

Bienvenue dans mon dépôt dédié à **Power BI**, où je regroupe mes projets d’analyse de données et de visualisation interactive.

Ce repository met en avant ma capacité à transformer des données brutes en **insights clairs, exploitables et visuellement impactants** à travers des dashboards professionnels.


## 📁 Structure du Repository

Chaque projet contient :

* Fichiers `.pbix`
* Jeux de données (CSV / Excel)
* Documentation des analyses


## 🛠️ Technologies & Concepts Utilisés

Les projets couvrent les fonctionnalités clés de Power BI :

* **Power BI Desktop**
* **Power Query (ETL & Data Cleaning)**
* **Modélisation des données (Data Modeling)**
* **DAX (Data Analysis Expressions)**
* **Création de dashboards interactifs**
* **Visualisations avancées (charts, KPIs, slicers)**
* **Relations entre tables**
* **Mesures et colonnes calculées**


## 📊 Compétences Démontrées

- ✔️ Nettoyage et transformation des données (ETL)
- ✔️ Création de modèles relationnels efficaces
- ✔️ Conception de dashboards interactifs et dynamiques
- ✔️ Utilisation avancée de DAX pour analyses métier

## ▶️ Comment Utiliser les Projets

1. Télécharger Power BI Desktop

2. Ouvrir un fichier `.pbix`

```bash
File → Open → project.pbix
```

3. Explorer :

* Les dashboards interactifs
* Les filtres et slicers
* Les relations de données
* Les mesures DAX


## 📌 Exemple de Workflow

1. Import des données (Excel / CSV)
2. Nettoyage avec **Power Query**
3. Modélisation des tables
4. Création de mesures DAX
5. Conception du dashboard
6. Analyse et interprétation
